# Lesson 07 – Async / asyncio

## Topics
- `async def` / `await`
- `asyncio.run()` entry point
- `asyncio.gather()` — concurrent tasks
- `asyncio.sleep()` — simulate I/O
- `asyncio.create_task()` — fire and forget
- `asyncio.timeout()` — per-operation timeout (Python 3.11+)
- `asyncio.Queue` — producer / consumer pattern
- Async context managers (`async with`)
- Async generators (`async def … yield`)

## Run

```bash
python main.py
python exercise.py
```
