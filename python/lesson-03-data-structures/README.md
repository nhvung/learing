# Lesson 03 – Data Structures

## Topics
- `list`: append, insert, remove, sort, slice, `*` unpacking
- `tuple`: immutable, named tuples (`collections.namedtuple`, `typing.NamedTuple`)
- `dict`: creation, `.get()`, `.items()`, `.update()`, dict merge (`|`)
- `set`: union, intersection, difference, symmetric difference
- List, dict, set comprehensions
- `collections`: `Counter`, `defaultdict`, `deque`
- `heapq` for priority queue
