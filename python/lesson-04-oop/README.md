# Lesson 04 – OOP & Dataclasses

## Topics
- `class`, `__init__`, `self`
- `@property` and `@property.setter`
- `@classmethod` and `@staticmethod`
- `__repr__`, `__str__`, `__eq__`, `__lt__`, `__hash__`
- `@dataclass` — auto-generates `__init__`, `__repr__`, `__eq__`
- `field()`, `__post_init__`
- `@dataclass(frozen=True)` — immutable record
- Class variables (`ClassVar`)
- Inheritance, `super()`
