# Lesson 01 – Introduction & Types

## Topics
- Top-level statements (no `Main` boilerplate)
- Built-in value types: `int`, `double`, `bool`, `char`
- `string` interpolation and verbatim strings
- `var` type inference
- Nullable types (`int?`, `string?`)
- Constants and enums
- `Console.Write` / `Console.WriteLine`
