# Lesson 05 – Interfaces & Generics

## Topics
- `interface` definitions and implementation
- Explicit interface implementation
- Default interface methods (C# 8+)
- Generic classes and methods (`<T>`)
- Generic constraints (`where T : class`, `where T : new()`, etc.)
- `IEnumerable<T>`, `IComparable<T>`, `IEquatable<T>`
- Covariance and contravariance (`out T`, `in T`)
- `IDisposable` and `using` statement
