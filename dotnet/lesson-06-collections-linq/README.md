# Lesson 06 – Collections & LINQ

## Topics
- `List<T>`, `Dictionary<K,V>`, `HashSet<T>`, `Queue<T>`, `Stack<T>`
- Collection initializer syntax
- LINQ query syntax vs method syntax
- Common LINQ: `Where`, `Select`, `OrderBy`, `GroupBy`, `First/Last`, `Any/All`, `Count`, `Sum/Average/Min/Max`
- `SelectMany` (flatten)
- Deferred execution
- `ToList()`, `ToArray()`, `ToDictionary()`
- `IEnumerable<T>` as the universal sequence abstraction
- `Span<T>` brief introduction
