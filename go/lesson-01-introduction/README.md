# Lesson 1 — Introduction & Syntax

Covers the fundamental building blocks of Go: program structure, variables, types, and control flow.

## Key Concepts

- `package main` + `func main()` — entry point of every executable
- Variable declaration: `var`, `:=` (short declaration), `const`
- Zero values — every type has a default value
- Basic types: `int`, `float64`, `bool`, `string`, `byte`, `rune`
- Control flow: `if/else`, `for` (only loop in Go), `switch`, `defer`

cd src
go run .
