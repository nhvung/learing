# Lesson 4 — Structs & Interfaces

Covers Go's type system: struct definition, methods (value vs pointer receivers), interfaces, embedding, and type assertions.

## Key Concepts

- Struct definition and initialization
- Value vs pointer receivers — when to use each
- Constructor functions (`New...`)
- Interface implicit satisfaction — no `implements` keyword
- Struct embedding — promotes fields and methods
- Type assertion and type switch
