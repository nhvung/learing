# Lesson 3 — Data Structures

Covers Go's built-in collection types: arrays, slices, and maps, plus the `sort` package.

## Key Concepts

- **Array** — fixed size, value type (copied on assignment)
- **Slice** — dynamic, reference type (`make`, `append`, `copy`, reslicing)
- **Map** — hash table (`make`, read/write, delete, range iteration)
- **sort** package — sorting slices with custom comparators
