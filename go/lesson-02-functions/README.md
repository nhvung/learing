# Lesson 2 — Functions

Covers Go's function features: multiple return values, named returns, variadic functions, first-class functions, and closures.

## Key Concepts

- Multiple return values — idiomatic Go pattern for returning results with errors
- Named return values and naked returns
- Variadic functions (`...T`)
- Functions as values and parameters (higher-order functions)
- Closures — functions that capture their surrounding scope
- `defer` with function values
