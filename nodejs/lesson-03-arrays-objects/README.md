# Lesson 3 — Arrays & Objects

Covers JavaScript's built-in data structures and the functional array methods that are used everywhere in Node.js code.

## Key Concepts

- Array methods: `map`, `filter`, `reduce`, `find`, `some`, `every`, `flat`, `sort`
- Object methods: `Object.keys/values/entries`, `Object.assign`, `Object.freeze`
- Destructuring: array and object patterns with defaults and renaming
- Spread (`...`) for shallow copy and merge
- Computed property names and shorthand syntax

```bash
node src/main.js
```
