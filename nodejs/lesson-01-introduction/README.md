# Lesson 1 — Introduction & Syntax

Covers the foundational building blocks of JavaScript in Node.js: variables, data types, type coercion, and control flow.

## Key Concepts

- `const` / `let` (avoid `var`) — block scope, temporal dead zone
- Primitive types: `string`, `number`, `boolean`, `null`, `undefined`, `symbol`, `bigint`
- `typeof`, `===` vs `==` — always use strict equality
- Falsy values: `0`, `''`, `null`, `undefined`, `NaN`, `false`
- Template literals, optional chaining `?.`, nullish coalescing `??`
- `if/else`, `for`, `for...of`, `switch`
