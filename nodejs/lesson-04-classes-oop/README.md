# Lesson 4 — Classes & OOP

Covers ES6+ class syntax, inheritance, private fields, getters/setters, static methods, and composition patterns.

## Key Concepts

- `class` syntax — syntactic sugar over prototype-based inheritance
- `constructor`, `super()`, `extends`
- Private fields with `#` — truly private (ES2022)
- `get` / `set` accessors
- `static` methods and properties
- `instanceof` for type checking
- Mixins — composing behavior without deep inheritance chains
