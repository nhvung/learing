# Lesson 7 — File I/O & Events

Covers Node.js file system operations, the `path` module, and event-driven programming with `EventEmitter`.

## Key Concepts

- `fs/promises` — promise-based file I/O (preferred over callbacks)
- `path.join`, `path.resolve`, `path.basename`, `path.extname`
- `__dirname` / `__filename` — current file location (CommonJS only)
- `EventEmitter` — the backbone of Node.js's event-driven architecture
- Streams (brief intro) — readable and writable

## How to Run

```bash
node src/main.js
```
