# Lesson 2 — Functions

Covers every function form in modern JavaScript: declarations, arrow functions, closures, higher-order functions, and the rest/spread operators.

## Key Concepts

- Function declaration vs expression vs arrow function
- Arrow functions have no own `this` — important for class methods and callbacks
- Default parameters, rest parameters (`...args`), spread (`...arr`)
- Closures — the function remembers the scope it was created in
- Higher-order functions — functions that take or return functions
- IIFE (Immediately Invoked Function Expression)
