# Lesson 10: Preparing for Java Certification and Careers

## Java Certifications

| Certification | Level | Topics |
|---------------|-------|--------|
| **Oracle Certified Foundations Associate (OCFA)** | Beginner | Syntax, OOP basics |
| **Oracle Certified Associate (OCA) — Java SE 11** | Intermediate | Full language, OOP, APIs |
| **Oracle Certified Professional (OCP) — Java SE 17** | Advanced | Streams, modules, concurrency |

## OCP Exam Key Topics (Java SE 17)
- Java basics, operators, and flow control
- OOP: classes, interfaces, inheritance, polymorphism
- Generics and collections
- Functional programming: lambdas, streams, `Optional`
- Exception handling and assertions
- Java I/O and NIO.2 (`java.nio.file`)
- Concurrency: `Thread`, `ExecutorService`, `CompletableFuture`
- Modules (`module-info.java`)
- JDBC

## Study Strategy
1. **Work through this course** — it covers the major OCP topics
2. **Practice questions**: Enthuware, Sybex OCP Study Guide
3. **Code daily**: the exam tests real code knowledge, not memorization
4. **Mock exams**: simulate 3-hour sessions with timed practice tests

## Career Paths
| Role | Skills Needed |
|------|---------------|
| Backend Developer | Spring Boot, REST, SQL, JPA/Hibernate |
| Android Developer | Java/Kotlin, Android SDK |
| DevOps / Platform | Java internals, Docker, CI/CD |
| Data Engineer | Java + Spark/Kafka, big-data pipelines |
| Enterprise Architect | Design patterns, microservices, cloud (AWS/GCP) |

## Key Java Topics to Master for Jobs
- Spring Boot (most common Java framework)
- SQL + JPA/Hibernate (persistence)
- REST API design
- Docker & Kubernetes basics
- Git and CI/CD (GitHub Actions, Jenkins)
- Unit testing (JUnit 5 + Mockito)
- Concurrency and thread safety

## Practice Challenge
Without looking at notes, write from memory:
1. A class that implements `Comparable<T>`
2. A lambda that filters and maps a `List<String>` using streams
3. A `try-with-resources` block with a custom `AutoCloseable`
4. A `CompletableFuture` chain that fetches two values in parallel and combines them

If you can do all four — you're exam-ready.
